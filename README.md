Задание:
28.Карточки животных — вывод карточек с названием и фото животного.
Функционал:
В приложении показано 8 карточек животных. При наведении курсора на карточку, появялется описание животного, а фон затемняется. Это работает на всех карточках. Также на всех карточках справа вверху изображено сердце, при нажатии на него карточка становится в "избранном", а сердце становится красным. При повторном нажатии сердце обратно становится серым, и карточка удаляется из "избранного".
